Did I...? - A history of the project

Honestly? This project was born in an elevator, where I wasn't sure if I locked the car. I decided to develop an App for the iPhone where you could very easily thanks to shortcuts! But guess what? I'm not that familiar with Swift or SwiftUi for that matter, and I kept getting stressed because things didn't work. Story of many developers lives, huh? Anyways, I kept iterating until I got to a point that it was pretty good and I decided to send it to the Swift Challenge. However, I wanted to do something better (kinda) so I opened my laptop and started coding in C, my favorite language by far (and the one I have more experience with, true). And... That's how this project was born!

The GitHub page is https://github.com/Intron014/did-i, and I hope you like it :)
